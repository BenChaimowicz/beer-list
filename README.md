## Beer List By Ben Chaimowicz
## React

## To run this project please:

## Clone this repository with GitHub CLI:
### `gh repo clone BenChaimowicz/beer-list`
### `cd beer-list`
### `npm run build`
### `npm i -g serve`
### `serve -s build`

Open your browser and go to [http://localhost:3000](http://localhost:3000)
